#   python虚拟环境管理

-   ##   windows系统中默认是没有安装Python的，首先需要安装Python2.7与Python3.6
-   ##  安装Python2.7版本的虚拟环境 env-py2：
    -   `virtualenv -p C:\Python27\python.exe [name]`
-   ##  安装Python3.6版本的虚拟环境env-py3:
    -   `virtualenv -p C:\Python36\python.exe [name]`

-   ##  启动虚拟环境
    -   进入安装后的`scripts`目录
    -   `activate`激活
